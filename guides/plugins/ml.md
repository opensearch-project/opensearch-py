### ML Commons Plugin

You can use the [ML Commons Plugin API](https://docs.opensearch.org/latest/ml-commons-plugin/api/index/) to manage machine learning models, agents, connectors, and memory for AI-powered search. The plugin supports model lifecycle management, conversational AI agents, and memory management for chat applications.

For detailed API reference, see the [opensearch-py API documentation](https://opensearch-project.github.io/opensearch-py/api-ref).
